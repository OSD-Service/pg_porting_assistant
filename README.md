# pg_porting_assistant
Automatic porting assistant for Oracle to PostgreSQL.

## env preset
#### Directory tree 
```bash
- pg_porting_assistant
- fastagent.config.yaml
- fastagent.secrets.yaml # optional
- ASIS
  |- test

  |- test2
  |  |- test2.sql
  |  |- test3.sql
```

## Usage
```bash
./pg_porting_assistant
```
